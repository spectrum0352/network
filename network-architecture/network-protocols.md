
# Network Protocols

- Communication Rules: These are the sets of rules and standards that govern how devices communicate and exchange data on a network, such as:

- IP: The fundamental protocol that defines how data is addressed and routed across the internet.

- TCP: A protocol that ensures reliable data delivery by establishing connections, managing data flow, and handling errors.

- UDP: A protocol that provides faster data transfer without guaranteed delivery, often used for real-time applications like video streaming.

- HTTP: The protocol used for web browsing, governing how web servers and browsers communicate to exchange web pages.

- DNS: A system that translates domain names into IP addresses, allowing users to access websites without memorizing numerical IP addresses.
