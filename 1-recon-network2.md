# Network

* Details collected to map the network and identify accessible services:
* Domain Names: Internal and external domain names
* IP Address Ranges: Public and internal network blocks, IPs of reachable systems
* Live Hosts: Reachable or active systems in the network
* Rogue or Hidden Websites: Private/internal web portals or shadow IT
* Network Services: Running TCP/UDP services and exposed ports
* Access Controls: Mechanisms like ACLs (Access Control Lists)
* Security Systems: Active IDS/IPS (Intrusion Detection/Prevention Systems)
* VPN Access Points: Entry points for remote access
* Communication Protocols: Protocols in use (e.g., HTTP/S, SMB, RDP, etc.)
* Authentication Methods: Password, token, certificate-based, etc.
* Phone Numbers: Analog/digital contacts related to infrastructure
