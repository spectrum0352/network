# Types: 

SYN Scan, ACK Scan, XMAS Scan.

# Security Solutions: - 

Security Solution: Firewalls, Intrusion Detection/Prevention Systems (IDS/IPS), Regular Security

Audits.

# Detection: - 

Detection in SIEM: Unusual connection attempts to various ports, repeated scanning activities. - SIEM Solution Features: Log analysis, correlation of events, and real-time alerting for suspicious port

scanning activities.

# Mitigation: - - - 

Configure firewalls to block or rate limit suspicious scanning activities.

Regularly audit and close unnecessary open ports.

Implement intrusion detection/prevention systems to detect and block port scanning.

# Nmap SYN Scan: 

Example: An attacker uses Nmap to perform a SYN scan, identifying open ports on a target system and

potential vulnerabilities.
